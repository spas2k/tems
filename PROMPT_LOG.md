# TEMS — Design & Feature Prompt Log

> This file tracks every design and feature prompt used to build and evolve the TEMS application.  
> **Only design/feature prompts are logged here** — troubleshooting, server starts, and debugging prompts are excluded.

---

## How to Read This Log

Each entry contains:
- **#** — Sequential prompt number
- **Date** — When the prompt was given
- **Prompt** — The exact or summarized prompt text
- **Summary** — Brief description of what was built or changed
